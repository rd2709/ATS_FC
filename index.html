Google Sheets sync — setup guide
1
Go to drive.google.com → New → Google Sheets → name it "First Connect ATS"
2
Click Extensions → Apps Script. Delete all existing code. Paste the script below.
3
Click Save → Run → doGet → click Authorize and allow permissions.
4
Click Deploy → New Deployment → Web App. Set: Execute as Me, Access Anyone. Click Deploy.
5
Copy the Web App URL → paste it in the sync panel in the left sidebar → click Sync now.
